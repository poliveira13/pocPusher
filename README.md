# Poc Pusher


### React Native
-> Yarn (para instalar das libs android)
-> cd ios -> pod install (para instalar as libs no ios)

-> Para buildar (dev)
```bash
$ yarn start
```

-> Emulador Ios
```bash
$ yarn ios
```
 OU
 
-> Emulador Android
```bash
$ yarn android
```
